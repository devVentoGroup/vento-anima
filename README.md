# ANIMA Mobile

Aplicación móvil nativa de asistencia y control de acceso con check-in/check-out en tiempo real.

## Características

- Splash screen animado
- Autenticación con email y contraseña
- Check-in/Check-out
- Seguimiento de horas trabajadas
- Diseño elegante en porcelana y rosa eléctrico

## Stack

- React Native + Expo
- Expo Router (navegación)
- Reanimated 3 (animaciones)
- Context API (autenticación)
- TypeScript

## Instalación

```bash
cd apps/anima-mobile
pnpm install
```

## Desarrollo

```bash
pnpm run start
```

## Compilar

```bash
pnpm run build
